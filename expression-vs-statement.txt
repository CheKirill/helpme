выражение (expression) -- то, что можно вычислить:
    + (операция, а не оператор!)
    -
    *
    /
    ==
    not
    and
    or
    f( )
    [ список ]
    1

оператор (statement) -- инструкция, что делать:
    =
    if
    for
    class
    import

Функциональное программирование:
 * всё выражение, нет операторов
 * функции без побочных эффектов
 * функция -- полноценный объект
